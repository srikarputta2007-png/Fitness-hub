# Fitness Hub - React + JavaScript + Vite

A front-end fitness management application based on the uploaded FEDF abstract. It collects user information, calculates BMI, estimates calories, generates workout and diet recommendations, includes routing, local storage, progress tracking, responsive design, and a workout timer.

## Run Project

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Main Features

- React + Vite + JavaScript
- React Router navigation
- BMI calculator and calorie estimation
- Fitness goal recommendation engine
- Workout schedule and diet plan pages
- LocalStorage profile and progress saving
- Workout timer
- Responsive UI and form validation
